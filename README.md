# HEJTERI Clan Website

A premium dark-mode clan website built with Next.js App Router, TypeScript, and Tailwind CSS.

## Stack

- Next.js 16.2.1
- React 19.2.4
- TypeScript 6.0.2
- Tailwind CSS 4.2.2

## Run

```bash
npm install
npm run dev
```

## Pages

- `/` home
- `/roster` roster with copy-to-clipboard actions
- `/about` clan story and timeline
